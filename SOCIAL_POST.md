## LinkedIn (longer)

**I just rebuilt my personal portfolio from the ground up — here's what's new.**

What started as a simple profile site has become something I'm genuinely proud of. The latest update adds:

**Clickable experience & project cards** — each role and project now has its own dedicated page with a split layout: full story on the left, rotating image gallery on the right. Every page has a shareable URL.

**Story Series** — I can now group stories into episodic series (think 90-day challenges, monthly reflections, or themed journals). Visitors browse the series, click into episodes, and read the full story with media.

**External media support** — Google Drive links, YouTube, Vimeo, and any public image URL can be used as media. Google Drive sharing links auto-convert to direct-access format.

**AI-enhanced content** — Preview text and titles are generated with Claude (Haiku), giving each card a compelling hook that makes visitors want to click through.

**Full media management** — upload files, paste external links, replace images, and control gallery rotation speed — all from the dashboard.

This was built entirely with:
- **Next.js 16** (App Router, server components)
- **Supabase** (Postgres, Auth, Storage)
- **Framer Motion** for animations
- **Tailwind CSS 4** for styling
- **Anthropic Claude API** for AI features

The codebase is growing but staying clean — everything is componentized and the data model is flexible enough to handle whatever content type I want to add next.
